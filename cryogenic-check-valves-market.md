<p><p>Cryogenic check valves are available in different types depending on the actuation mechanism: Manual type, Electric type, Pneumatic type, and Hydraulic type. Manual type valves require manual operation to open or close the valve. Electric type valves use an electric actuator for control. Pneumatic type valves rely on compressed air to operate. Hydraulic type valves use hydraulic pressure to control the valve. Each type has its own advantages and is suitable for different applications in cryogenic systems.</p></p>
